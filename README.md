# CCS Attendance System

**MSU-IIT CCS TABCOM 2025-2026**

A QR-code based attendance monitoring system for the College of Computer Studies.

## Tech Stack
- **Frontend**: React (Vite)
- **Backend**: Node.js, Express
- **Database**: PostgreSQL (Supabase) via Prisma ORM
- **Hosting**: Railway

## Deployment
This project is deployed on **Railway**.
- **Backend**: Connects to Supabase PostgreSQL.
- **Frontend**: Connects to the Railway backend URL.

## Local Development
1. Clone the repo.
2. Setup `backend/.env` with `DATABASE_URL`.
3. Run `npm install` in both `frontend` and `backend`.
4. Run `npm start` / `npm run dev`.
